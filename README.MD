# Edulend - Sistema de Gestión de Préstamos Educativos 🎓💻

Edulend es una aplicación de consola desarrollada en Python diseñada para centralizar y simplificar el préstamo de recursos tecnológicos (portátiles, tablets, libros) en entornos educativos.

## 🚀 Características
- **Registro y Autenticación:** Sistema de usuarios con almacenamiento en diccionarios.
- **Dashboard de Usuario:** Panel interactivo para solicitar artículos y consultar el estado de préstamos en tiempo real.
- **Panel Administrativo:** Acceso restringido para la aprobación y gestión de solicitudes.
- **Persistencia Temporal:** Gestión de datos estructurada mediante listas y diccionarios.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python
- **Versionamiento:** Git & GitHub (Metodología Git Flow)
- **Entorno:** Virtualenv

## 📋 Requisitos Previos
Asegúrate de tener instalado Python 3.10 o superior.
